# id-photo-studio

一个纯前端证件照处理工具，所有图片处理都在浏览器本地完成。

## 功能特性

- 支持点击选择图片和拖拽上传图片。
- 上传后展示原图预览，最大显示尺寸限制为 400px，并保持原始比例。
- 使用 `@imgly/background-removal` 在浏览器本地完成 AI 抠图。
- 抠图过程中显示加载状态和首次模型下载提示。
- 抠图完成后展示透明背景结果，并使用棋盘格背景辅助查看透明区域。
- 支持选择白色、标准蓝、标准红、渐变蓝、浅灰 5 种证件照底色。
- 支持一寸、二寸、小一寸、大一寸、签证照、社保照 6 种 300dpi 尺寸规格。
- 支持实时成品预览，并按当前尺寸和底色下载 JPG 照片。
- 抠图失败时展示用户友好的错误提示。

## 技术栈

- 原生 HTML
- 原生 CSS
- 原生 JavaScript
- Vite
- `@imgly/background-removal`
- `jszip`

## 本地运行

```bash
npm install
npm run dev
```

运行后打开终端中显示的本地地址，一般是 `http://localhost:5173`。

## 隐私声明

本项目的照片读取、预览和抠图流程都在浏览器本地完成，照片文件不会上传到任何业务服务器。

首次使用 AI 抠图时，浏览器会从 `@imgly/background-removal` 默认资源地址下载模型和 WASM 文件。下载的是运行所需模型资源，不是上传用户照片。模型资源会被浏览器缓存，后续使用通常会更快。
